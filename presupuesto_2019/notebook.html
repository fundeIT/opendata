Carga de los datos¶

Los datos fueron proporcionados por el Ministerio de Hacienda en archivos Excel, uno para el presupuesto propuesto y otro para el aprobado. En esta sección se cargan los datos y se integran en un solo dataset.

def nivel(institucion):
    if institucion / 100 == institucion // 100 :
        return '1'
    else:
        return '2'
In [59]:
indexes = ['ejercicio', 'area', 'institucion', 'unidad', 'linea', 'fuente', 'economico', 'objeto']
